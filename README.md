# How to use JSDoc

Contains JS source files, along with jsdoc configuration to illustrate usage of JSDoc. An explanation is in my blog post at [akshatsharma.com](http://www.akshatsharma.com/posts/jsdoc-howto/).

## To use

```
$ npm i -g jsdoc
$ jsdoc -c jsdoc.conf
```

Or,
```
$ npm i
$ npm run doc
```
